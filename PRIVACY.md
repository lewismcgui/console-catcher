# Privacy Policy — Console Catcher

**Last updated:** 2026-03-24

## What data is collected

Console Catcher captures JavaScript error messages (console.error, uncaught exceptions, unhandled promise rejections) from web pages you visit. Each error includes the error message, stack trace, page URL, line/column numbers, and timestamp.

## Where data is sent

All captured error data is sent **only to localhost (127.0.0.1:3777)** on your own machine. No data is ever sent to any external server, cloud service, or third party.

## What data is stored

Error data is stored in memory only (not written to disk) by the local MCP server. Data is lost when the server is stopped. Maximum 500 errors are retained at any time.

## What data is NOT collected

- No personally identifiable information
- No browsing history
- No cookies or authentication tokens
- No analytics or telemetry
- No data is sold or shared with third parties
